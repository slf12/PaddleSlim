paddleslim.common.cached_reader(reader, sampled_rate, cache_path, cached_id)¶

Sample partial data from reader and cache them into local file system. :param reader: Iterative data source. :param sampled_rate: The sampled rate used to sample partial data for evaluation. None means using all data in eval_reader. default: None. :type sampled_rate: float :param cache_path: The path to cache the sampled data. :type cache_path: str :param cached_id: The id of dataset sampled. Evaluations with same cached_id use the same sampled dataset. default: 0. :type cached_id: int
